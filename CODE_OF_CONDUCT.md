# Code of Conduct

Shadow follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

- Be respectful. Disagreement is fine; contempt is not.
- Ground arguments in code, citations, or measurements. Not in identity.
- Shadow ships into regulated banks. Sloppiness costs real people real money. Take the work seriously.
- No spam, no harassment, no political fight-picking in issues or PRs.

## Scope

This code of conduct applies to all repository spaces: issues, pull requests, code comments, commit messages, GitHub Discussions if/when enabled, and any in-person or video interactions tied to the project.

## Enforcement

Email the maintainer privately (see GitHub profile). Reports will be reviewed within 7 days. Consequences range from a private warning to a permanent ban from the project depending on severity.
