Record this on the Mac (Unity Play Mode or the desktop web client) as the fail-safe if the device
or projector misbehaves. Deterministic + offline — no live model.

## Shot list (target ~75 s)
1. (0:00) Banking · READY — show the loan document + Trust Bar + one status row + FIXTURE MODEL label.
2. (0:10) Type "show the source supporting the highest-risk finding" → grounded answer + citation
   chip + the B0L1 source overlay highlights + `LAST ACTION: highlight_source — EXECUTED`.
3. (0:30) Click Verify → SEALED · VERIFIED. Click Tamper → TAMPERED. Click Reset.
4. (0:45) Switch to Data Science → "why was this model selected?" → GBM + AUC focus (no loan/DTI).
5. (0:60) Switch to Coding Agent → "which change fixed the duplicate EventSystem?" → diff/test focus.
6. (0:72) Presenter reset → Banking READY. End.

## Recording tips
- 1920×1080, 30 fps. QuickTime screen recording or Screen Studio.
- Airplane mode / no network (proves it needs none).
- No voiceover claims about Eye / 6DoF / RGB capture.
